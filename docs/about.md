# About

This repository is a personal collection of templates and boilerplates used for DevOps and software development projects. It includes configurations for various tools and frameworks that are frequently used in production environments.

## Purpose

The goal is to provide quick-start templates that follow best practices and can be adapted for specific use cases. All templates are tested and maintained regularly.

## Contents

- **Terraform**: Infrastructure as Code for AWS, Azure, DigitalOcean, and more
- **Kubernetes**: Production-ready manifests for platform services
- **Docker Compose**: Complete stacks for development and production
- **Ansible**: Automation playbooks for server management
- **GitHub Actions**: CI/CD workflows
- **Microservices**: Spring Boot and Quarkus templates
